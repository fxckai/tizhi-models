# Tizhi Skill · 体制内思维模型

一个帮你理解体制运行逻辑、评估自身处境、在体制内保护自己的 **AI 思维模型**。  
适用于Openclaw等支持Skill机制的AI工具。

## 简介

体制的设计目标从来不是让人自我实现，而是 **不出错**。  
这个Skill会让你时刻警惕一句隐形的真话：  
> “我正在被塑造成什么？”

它并不教你投机钻营，而是教你——**在知道代价的前提下做出选择**。

## 安装

1. 将 `skill.md` 文件（本仓库中的 Markdown 格式文件）下载到你的项目中。
2. 对于 **Openclaw**，将文件放入项目下的 `.openclaw\workspace\skills` 目录。  
   对于其他AI工具，请参照其Skill/指令导入功能，加载该文件内容。
3. 在你的提示词或对话中激活该Skill：
   - 输入关键词 **“体制内”“编制”“公务员”“国企”“晋升”“官场”** 等，部分工具会自动唤起。
   - 或明确给出指令：“/skill tizhi”等。

## 核心内容预览

### 三大思维模型

- **系统 > 市场**  
  「做事」和「做人」的权重，在这里是倒过来的。

- **程序正义 > 结果正义**  
  程序的最大功能，是用来在追责时切割自己。

- **利益包装**  
  把道理讲对没用，得让听的人觉得这道理对他有用。

### 决策五问

做任何决定前，按序问自己：

1. 程序合规吗？
2. 风险能切割吗？
3. 领导会满意吗？
4. 这事能等吗？
5. 我的意图包装成他的利益了吗？

## 使用示例

激活Skill后，你可以这样提问：

- “我刚进市直单位，为什么每天都是填表开会，感觉没创造价值？”
- “领导把一个明显会出问题的项目交给我，怎么才能不背锅？”
- “同事推活给我，我该怎么拒绝又不得罪人？”

Skill会以冷静、通透的语气，从程序合规、风险切割、立场包装等角度帮你拆解。
